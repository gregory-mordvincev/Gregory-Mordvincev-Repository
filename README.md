# Sales Analysis Project

## 📊 Описание
Анализ продаж на основе таблицы orders.

## 🛠 Используемые технологии
- SQL

## 📌 Что было сделано
- Выручка по клиентам
- Топ-3 товара по выручке
- Выручка по месяцам
- Средний чек по клиентам
- Анализ клиентов с выручкой > 1000

## 📈 Основные выводы
- Основную выручку приносит клиент Maria
- Самый прибыльный товар — Laptop
- В январе выручка выше, чем в феврале
- Есть клиенты с высокой суммой покупок (>1000)
